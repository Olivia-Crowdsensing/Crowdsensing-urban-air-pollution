The folder contains several folders that have different meaning.

- 01-data-raw: this is generated data, with the **format is same as the gcn-Kriging requirements**. 
- 02-data-reshape-py: this is the python format that transformed the data format to the *R suitable* format.
- 03-data-reshape-R: R processed
- 03-data-reshape-R-variograms: this contains the variogram that generated from R-spatiotemporal-Kriging.
